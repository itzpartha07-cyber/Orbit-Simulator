# Orbit Simulator

A simple 2D orbit simulator written in Python to visualize gravitational interactions between celestial bodies. The project uses basic Newtonian mechanics to calculate gravitational forces and update the motion of bodies over time.

## Features

* Simulates gravitational attraction between multiple bodies.
* Real-time 2D visualization of orbits.
* Modular code structure for physics calculations and body definitions.
* Easy to modify initial conditions and experiment with different systems.

## Files

* `main.py` – Runs the simulation and handles the main loop.
* `body.py` – Defines the properties and behaviour of celestial bodies.
* `physics.py` – Contains the gravitational and motion calculations.
* `constants.py` – Stores simulation constants and parameters.

## Requirements

* check requirements.txt

Install dependencies with:

```bash
pip install -r requirements.txt
```

## Running the Simulator

```bash
python main.py
```

This project was created as a small learning exercise to better understand orbital mechanics, numerical simulation, and Python-based visualisation of physical systems.
